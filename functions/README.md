# FairShare Cloud Functions (Security Mutations)

This folder contains trusted backend mutations used by the Android app:

- `sendNudge`
- `confirmSettlement`

## Local setup
1. Install Firebase CLI.
2. From this directory, run `npm install`.
3. Start emulator with `npm run serve`.

## Deployment
Run `npm run deploy` from this directory after authenticating Firebase CLI and configuring the project.
